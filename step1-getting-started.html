How new are you?
Using waffles and understanding the differences and advantages of it is going to require some CSS knowledge on your part. Familiarity with the CSS3 box model widths, heights, centering an element, margin vs. padding, positioning, and floating are going to be necessary.
If those ideas sound foreign to you, do not despair! Check out the Step 0 - I'm new! page for some resources that we think are pretty good.

Initializing Waffles
To get started with waffles, give your body or top-most element the class of waffles.
Doing so resets margin, floating, and removes padding.
Let's create some columns
Waffles subdivides a container into 12 potential columns. This number was picked because it's evenly divided 2, 4, 3, and 6. It's most likely that your layout needs will be covered by some sum of these dimensions. More on that later.
Each width starts with a w- and ends with the number of the size you want. A whole row may add up to 12 before elements are wrapped to the next block line.
Use those browser dev tools to inspect the box model, margins, and sizes this creates!
